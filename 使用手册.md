# Focus 使用手册

专注学习，快速记下走神念头

---

## 快速开始

### 1. 添加到主屏幕（像 App 一样用）
- **iOS**: Safari 打开 → 点击分享 → "添加到主屏幕"
- **Android**: Chrome 打开 → 菜单 → "添加到主屏幕"

### 2. 首页专注
- 输入当前学习任务（如"高数复习"）
- 点击"开始专注"启动番茄钟（25分钟）

### 3. 记录走神念头
学习时突然想到其他事？
1. 点击底部 **「记一下」**
2. 快速写一句话（如"查明天考试时间"）
3. 选择类型标签（杂念/待查/任务/情绪/灵感）
4. 选择提醒时间，点击保存
5. 自动返回专注页面，继续学习

### 4. 回看处理
到提醒时间了：
1. 打开「回看箱」标签
2. 对每条记录选择操作：
   - ✅ **已处理** - 标记完成
   - ⏰ **延后** - 30分钟后再提醒
   - 📦 **归档** - 不需要处理了

### 5. 查看统计
「统计」页面查看：
- 今日/本周走神次数
- 处理率
- 走神类型分布（杂念、待查、任务等）
- 高频走神时间段

---

## 核心原则

- **记录要快** - 3秒内完成，不中断学习
- **先记后做** - 把念头存起来，专注完再处理
- **定期清空** - 晚上统一处理，不留心理负担
